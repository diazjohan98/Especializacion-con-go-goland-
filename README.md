# 🚀 Especialización en Google Go (Golang) - UC Irvine

Este repositorio documenta mi progreso, notas de estudio y el código fuente de los laboratorios prácticos desarrollados durante el **Programa Especializado: Programación con Google Go**, impartido por la Universidad de California, Irvine (UCI) a través de Coursera.

## 🎯 Objetivo del Repositorio

Como Ingeniero de Sistemas con bases sólidas en desarrollo Frontend, el propósito de este repositorio es evidenciar mi transición hacia el desarrollo Full-Stack y Backend avanzado. Aquí plasmo mi dominio progresivo sobre la sintaxis, las estructuras de datos, y los conceptos avanzados de Go.

## 📚 Estructura de Aprendizaje

El código en este repositorio está organizado siguiendo los módulos de la especialización:

1. **Fundamentos de Go:** Tipos de datos, variables, control de flujo y manipulación de strings/archivos.
2. **Funciones, Métodos e Interfaces:** Implementación de programación orientada a objetos adaptada a la filosofía de Go, uso de punteros y abstracción de datos.
3. **Concurrencia en Go (Próximamente):** Dominio del paralelismo utilizando Goroutines y Channels para la construcción de sistemas altamente eficientes y escalables.

## 🛠️ Entorno de Desarrollo

- **Lenguaje:** Go (Golang)
- **Entorno:** Visual Studio Code / Coursera Lab Sandbox
- **Herramientas de Go:** `go build`, `go run`, `go fmt`, y depuración con `Delve`.
